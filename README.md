# Crypto-Block

:warning: Code not compatible with system windows


## Require

- g++
- libjsoncpp
- libcrypto++-dev
- libcrypto++-utils
- libcryptos++-8

## Compile project

```sh
$ mkdir build -p .
$ cd build
$ CXX=/usr/bin/g++
$ cmake ..
$ make

```

## Run

```sh
$ ./cryptos
```

## Configure programme

```sh
$ nano ~/.crypto/wallet/ids
```
